# AI Value & ROI Optimizer (Next.js)

A ready-to-run Next.js (App Router) demo that visualizes AI program value/ROI, adoption, reliability, and includes a What‑If simulator. Uses Tailwind + lucide-react + Recharts + lightweight shadcn-style UI components (included).

## Quickstart

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Tech

- Next.js App Router
- Tailwind CSS
- Radix UI primitives (for Select, Tabs, Sheet, Slider, Switch)
- Recharts for charts
- lucide-react for icons

## Project layout

- `app/` – App Router pages (`/ai-value-roi` is the dashboard)
- `components/ui/` – minimal UI primitives (no external codegen needed)
- `lib/utils.js` – `cn` helper
- `tailwind.config.js` – Tailwind setup
- `postcss.config.mjs` – PostCSS config
- `next.config.mjs` – Next config
